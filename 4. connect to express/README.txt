Setting Up Webpack Dev Middleware in Express

What is webpack dev server?
Webpack dev server is a live reloading server for webpack.

What is webpack hot middleware?
this allow you to use hot reloading into an existing server without webpack-dev-server.

1. npm i webpack-dev-middleware webpack-hot-middleware
2. make a webpack.dev.config file
   a. make your path '/'
   b. publicPath:'http:localhostL3000

3. build express application
